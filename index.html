Lesson One

HTML

Hyper Text Markup Language, this is the main document type of the web, it glues everything together. HTML is made up of Text content this is what you see and Markup this is what it looks like. It also makes references to other documents, images and videos.



THE WEB
  • Servers- computers that host the files that makeup the web
  • Internet- the world’s largest computer network
  • Browsers- a program that runs on your computer to display files found on the web
  • URLS- How you refer to documents on the web.
  • HTTP- protocol of the web
  • Hyperlinks- links in between pages
  • It was developed in the 1990’s
  • 30 billion pages

TAGS AND ELEMENTS OH MY
  • Markup is made up of tags, tags are always contained within angled brackets most having opening and closing however some that are called void tags do not require closing tags.
  • Opening Tag Content Closing tag

COMPUTERS ARE JUST NOT SMART
  • Computers are not smart because they interpret instructions literally.
  • This makes them very unforgiving.
  • One small mistake will turn into huge problems

INLINE AND BLOCK
  • these are known as container Elements
  • Inline are span, a br, img and strong
  • Block is div, p, form,block elements form an invisible box around the content

Lesson Two

TWO LANGUAGES
  • HTML CSS
  • HTML- Controls the structure of a webpage
  • CSS- Cascading Style Sheets language is used for describing the look and formatting of a document written in a markup language.

    • HTML STRUCTURE
  • The Tree like structure of HTML comes from the fact that HTML elements can have other elements inside of them
  • DOM- Document Object Model when programmers talk about the “DOM” they are talking about the tree like structure of a page.

BOXES AND INDENTATIONS
  • Boxes, Boxes, Everywhere!!
  • Everything is in a box
  • The relationship between indented HTML and Boxes As you read an HTML document as text, you see a wave of changing indentations going up and down the left side of the document the more indented an element is the more deeply nested its corresponding box is.

TEXT EDITOR

When writing code programmers use special text editors. These editors make the programmers life easier for example some text editors will automatically generate a closing HTML when you write an opening tag.
